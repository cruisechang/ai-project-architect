# AGENTS

Documentation maintenance rules for this repository:

- When command names, workflow, or product positioning change, update every top-level `README*.md` in the same change.
- Do not update only `README.md` or only one locale.
- The current CLI commands are `init`, `prompt`, `list-skills`, `doctor`, and `version`.
- Repo-local skill names should follow the `apa-*` series for consistent naming.
- Prefer repo-local skills plus native repo commands such as `make test` and `go test ./...` in examples.
